# AI_Powered_Medical_Insurance
AIDI Capstone

## Steps to run this app
1. Pull the required docker images
docker pull smehra999/med-streamlit
docker pull smehra999/med-client

2. Start the system
docker compose up --build

MedStream (Streamlit App) → http://localhost:8501
FAPP Forecasting App → http://localhost:8502
React Frontend → http://localhost:5173
