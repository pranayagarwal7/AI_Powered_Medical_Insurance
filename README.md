# AI_Powered_Medical_Insurance
AIDI Capstone

## Steps to run this app
Open three terminals, one to run Medstream.py, one to run FAPP.py, one to run the front end client folder
run the following commands in order
1. streamlit run Medstream.py --server.port PORT NUMBER 
2. streamlit run FAPP.py --server.port PORT NUMBER  
(ensure port numbers are different for both steps 1 and 2)
3. cd client - npm run dev
4. Click the url inside the terminal after running npm run dev (Which will run inside client folder)
